=========================================================
       MANUAL DE USO - DEEPCODE AI (Para Iniciantes)
=========================================================

Este guia foi preparado para você utilizar o DeepCode de forma simples e direta.
Toda a instalação complexa já foi realizada e configurada automaticamente.

---------------------------------------------------------
1. COMO INICIAR O DEEPCODE
---------------------------------------------------------
Você tem duas opções fáceis:

OPÇÃO A: Área de Trabalho (Mais fácil)
   -> Procure o atalho "DeepCode AI" na sua Área de Trabalho.
   -> Dê um duplo clique nele.

OPÇÃO B: Pela Pasta do Projeto
   -> Abra a pasta "DEEPCODE" no seu Desktop.
   -> Dê um duplo clique no arquivo "INICIAR_DEEPCODE.bat".

O QUE VAI ACONTECER?
1. Uma janela preta (terminal) vai abrir. NÃO FECHE ELA!
2. O sistema vai carregar (pode demorar alguns segundos).
3. Seu navegador de internet (Chrome, Edge, etc.) vai abrir automaticamente com o DeepCode pronto para uso.
   (Endereço: http://localhost:8510)

---------------------------------------------------------
2. COMO USAR O DEEPCODE
---------------------------------------------------------
Na tela que abrir no navegador:
1. Digite sua pergunta ou tópico de pesquisa na caixa de texto.
2. O DeepCode vai usar Inteligência Artificial para pesquisar e responder.

NOTAS IMPORTANTES:
- As chaves de API (Gemini e Groq) JÁ FORAM CONFIGURADAS para você.
- O modelo padrão está definido como Gemini 1.5 Pro (Google).

---------------------------------------------------------
3. COMO PARAR O SISTEMA
---------------------------------------------------------
Quando terminar de usar:
1. Volte para a janela preta (terminal).
2. Clique nela e aperte as teclas 'Ctrl' e 'C' juntas.
3. Ou simplesmente feche a janela preta.

---------------------------------------------------------
4. SOLUÇÃO DE PROBLEMAS (Se algo der errado)
---------------------------------------------------------
- Se o navegador não abrir: Digite http://localhost:8510 na barra de endereço.
- Se aparecer erro de porta em uso: Reinicie o computador ou feche janelas de terminal antigas.
- Se precisar reinstalar: Execute o arquivo "install_deepcode.bat" na pasta do projeto.

---------------------------------------------------------
INFORMAÇÕES TÉCNICAS (Avançado - Opcional)
---------------------------------------------------------
- Python Version: 3.11 (Configurado via venv)
- Instalação: C:\Users\FABIO\DeepCode_Workspace
- Configuração: mcp_agent.config.yaml (Editável se quiser trocar chaves)
- Patch de Correção: Aplicado automaticamente para compatibilidade com Python 3.11.

Bom trabalho!
